# Developer Portfolio 💻🔥

A sleek, dark-themed, and fully responsive personal portfolio website designed to showcase developer tasks, technical architecture skills, and engineering milestones. 

Built with **HTML** and **Tailwind CSS**, this site implements a professional continuous deployment pipeline—automatically updating live on the web with every git push.

## 🚀 Live Demo
🔗 **https://jasonmyportfolio.netlify.app/**

---

## 🛠️ Core Tech Stack & Tools
*   **Frontend:** HTML5, Tailwind CSS (via CDN)
*   **Version Control:** Git & GitHub
*   **Hosting & Deployment:** Netlify (Automated CI/CD Pipeline)
*   **Editor:** Visual Studio Code

## 📂 Highlighted Engineering Tasks

The portfolio aggregates key technical milestones across multiple scopes and development areas:

### ⚙️ Systems & Advanced Engineering
*   **Transformer-Based Text Synthesis:** Fine-tuned structural language models on custom text corpora to generate contextually aware data streams.
*   **Conditional GAN Translation:** Implemented paired image-to-image translations leveraging specialized neural network layers and balancing adversarial loss functions.
*   **Latent Space Navigation:** Created image synthesis processing pipelines navigating deep feature mapping arrays.
*   **Probabilistic Matrix Modeling:** Built state-transition sequence maps using Markov Chains to dynamically calculate linguistic probability structures.

### 🗄️ Backend Architecture & Database Management
*   **Social Media Analytics Platform:** Engineered backend data distribution routes and tracking metrics designed for scalable dashboard management.
*   **Relational Data Modeling:** Architected highly optimized schemas using Node.js and Express to seamlessly parse, process, and query JSON/relational datasets.
